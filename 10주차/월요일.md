# 전송-레벨 보안

## 웹의 특성
- 편리한 웹브라우저는 내부적으로 매우 복잡한 소프트웨어로 구성
    - 보안 결함 존재 가능
    - 새로운 기술로 업그레이드
    - 다양한 보안 공격에 취약점 드러남
- 웹서버를 침입기반으로 사용
    - 지역 사이트 서버에 연결된 데이터와 시스템 접근 가능
- 웹서비스 주 고객을 보통 사용자

## Web 트래픽 보안 방법
- Web 보안을 제공하는 여러가지 방법 가능
- Web 보안 방법은 Web 응용 범위나 TCP/IP 계층상 웹의 상대적 위치에 따라 달라짐
    - 네트워크 층
